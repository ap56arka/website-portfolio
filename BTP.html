PROJECT DESCRIPTION

Teleoperated and Autonomous Vehicle

December 22, 2077   Arka Pal

I have started working on this project from my 4th year as a part of Bachelor thesis and I am still working on the same. The main features of this project is explained below.

     

1. Sending driving command to the remote vehicle: For remote operation the obvious first step is to send driving command remotely to the actual vehicle. This has been achieved using a multi-client server architecture where, the driving command from a Logitech G29 gear is send via internet to the electric vehcile. It has been ensured that this communication process is efficient and robust to latency of the internet connection.

2. Implementation of steering and acceleration command in the vehicle: To actually achieve the steering command the motor attached to a power steering is used. In this case the motor is a brushless DC motor. The speed of the motor is controlled by controlling input voltage of this motor. To make sure the desired angle is achieved a modified PID controller is used
