PROJECT DESCRIPTION

Teleoperated and Autonomous Vehicle

December 22, 2077   Arka Pal

I have started working on this project from my 4th year as a part of Bachelor thesis and I am still working on the same. The first challange I tackled was to send driving commands generated by a Logitech G29 simulator to the remotely operated vehicle.

     

The solution was achieved using a Multi-client server architecture As the servers are running in a virtual private server, we can drive this vehicle from anywhere in the world. All you need is just a Logitech G29 simulator and some softwares written by us!
